This work builds a cross-layer design framework for vector-symbolic computing that connects symbolic kernels, architecture choices, and hardware implementations for efficient cognitive computing acceleration.

The project analyzes co-design principles across the stack and proposes heterogeneous memory architectures that reduce latency by up to 86% and improve energy efficiency by 2.6x over homogeneous designs.

The resulting framework provides a memory-centric VSA design template and identifies open challenges for scalable, energy-efficient brain-inspired AI systems.
