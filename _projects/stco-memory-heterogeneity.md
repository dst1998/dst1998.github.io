This project develops a system-technology co-optimization framework that connects algorithmic representations with heterogeneous memory for edge systems.

The framework is validated across DNNs, LLMs, vector-symbolic computing, and optimization solvers. It includes memory-centric model partitioning and mapping on heterogeneous chiplets and characterizes memory units that integrate compute, storage, and adaptive processing.

The goal is to provide design guidelines for scalable, energy-efficient edge intelligence accelerators.
