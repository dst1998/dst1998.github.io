This project explores reconfigurable 2T oxide-semiconductor non-volatile gain-cell memory for in-memory switching between static and dynamic operations in LLM inference and continual learning.

The accelerator uses a 3D sandwich integration scheme that combines oxide-semiconductor logic and memory layers on CMOS. It supports reconfigurable FP/INT compute modes, system-technology co-design modeling, and 3D inter-tier dataflow optimization to improve utilization and reduce on-chip data movement.

The design achieved 32.9 TOPS/W energy efficiency and 102.4 TOPS/mm2 compute density, with 2.3x/3.2x gains over planar CMOS and DRAM baselines.
